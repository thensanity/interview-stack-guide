# Full-Stack Engineer Learning Path

Complete path covering all layers — 6-week plan.

| Week | Focus | Key Files |
|------|-------|-----------|
| 1 | React vs Next.js | `apps/react-spa/`, `apps/web/` |
| 2 | REST + GraphQL API | `apps/api/`, `packages/graphql/` |
| 3 | NoSQL + Patterns | `packages/db/`, `packages/validation/` |
| 4 | Events + Scenarios | `packages/events/`, `/scenarios` |
| 5 | CI/CD + Docker | `.github/workflows/`, `docker-compose.yml` |
| 6 | AWS + K8s + Review | `infrastructure/`, `docs/DECISION-MATRIX.md` |

## Daily Practice (30 min)
1. Pick one scenario from `docs/interview-guide/scenarios.md`
2. Trigger it at `/scenarios`
3. Walk through the answer out loud pointing at code
4. Check mobile reference: `docs/MOBILE-REFERENCE.md`

## Capstone Demo (15 min)
1. Architecture overview (`/architecture`)
2. Create product (REST) → event in feed (SSE)
3. Trigger traffic_spike + recover
4. Show decision matrix (`/decisions`)
5. Show patterns museum (`/patterns`)
6. Mention CI/CD, AWS, K8s deployment options
